# 🛡️ Vault DX Pro

**Vault DX Pro** is a high-performance, client-side password and application manager built for power users. It provides a sleek, "App Store" style interface for managing credentials, URLs, and encrypted technical notes using a local-first security model.

## 🚀 Key Features

* **Elite Vault Security**: Create and manage `.kdbx` compatible files directly in your browser.
* **A-Z Alpha-Filter**: Instantly isolate applications by their starting letter for rapid access.
* **Deep-Search Engine**: Real-time search that parses through titles, links, and hidden cryptographic notes.
* **Expanded Detail View**: Single-click on icons to open a non-truncated "Big Form" viewer for full note visibility.
* **Dynamic Organization**: 
    * **Tabs**: Draggable category tabs for high-level sorting.
    * **Folder Trees**: Unlimited nested folder support for complex hierarchies.
* **Adaptive View Modes**: Toggle between Standard Grid, Compact Grid, Rich List, and Minimal Text views.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Tailwind CSS
- **Icons**: Lucide-JS (High-fidelity vector icons)
- **Engine**: Vanilla JavaScript (ES6+)
- **API**: File System Access API (Native local file handling)

## 📖 How to Use

1.  **Initialization**: Open the `index.html` file. Choose **Initialize** to create a new vault or **Unlock** to load an existing `.kdbx` file.
2.  **Organization**: Create Categories (Tabs) for broad sectors (e.g., Work, Social) and Folders for specific projects.
3.  **Entry Management**:
    * Click **Add Entry** to store a new app link and icon.
    * **Single-Click** any icon to view the full details without text cutting.
    * Use the **Alphabet Bar** or **Search Box** to filter through 1000+ entries instantly.
4.  **Sync**: All changes are committed directly to your local file via the **Commit Sync** protocol.

## 🔒 Security Principles

- **Zero-Cloud**: No data is ever sent to a server. Your master key and data stay within your local file.
- **Recovery Protocol**: Includes a recovery question/answer challenge to retrieve the master key if forgotten.
- **Privacy First**: Designed with an "Incognito" aesthetic to ensure low-profile usage.
